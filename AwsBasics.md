# AWS Basics

## What is AWS?

Amazon Web Services (AWS) is a cloud computing platform provided by Amazon.
It offers on-demand computing power, storage, databases, and networking services.

---

## Why Use AWS?

- No need to buy physical servers
- Pay only for what you use
- Highly scalable
- Secure and reliable
- Used by startups and enterprises

---

## AWS Global Infrastructure

AWS has:

- Regions
- Availability Zones
- Edge Locations

### Region
A physical geographic location (example: ap-south-1).

### Availability Zone (AZ)
One or more isolated data centers inside a region.
